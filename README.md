# TORQUE Monorepo

Nigeria-first viral marketplace with crypto/fiat escrow and social commerce.

- Package manager: pnpm
- Build: Turborepo
- Language: TypeScript

## Getting Started

- Copy `.env.example` files and fill in values
- Install deps: `pnpm install`
- Run dev: `pnpm dev`
- Build: `pnpm build`
